Event target 与 currentTarget 的区别

2017.10.29 hysunny JavaScript  热度 ℃

  1. target: 触发事件的某个具体对象，只会出现在事件流的目标阶段，谁触发谁命中;

  2. currentTarget: 绑定事件的对象，可能出现在事件流的任意一个阶段（捕获、目标及冒泡）中;

  3. 只有当事件流处于目标阶段时，target和currentTarget才是一致的。
